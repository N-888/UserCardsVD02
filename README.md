# UserCardsVD02
Стильные АНИМИРОВАННЫЕ КАРТОЧКИ ПОЛЬЗОВАТЕЛЕЙ с ИНТЕРАКТИВНЫМИ элементами, АДАПТИВНОЙ версткой всей страницы.
- Адаптивный дизайн - карточки автоматически подстраиваются под размер экрана
- Плавные анимации с разной скоростью для разных элементов:
o	Подъем карточки
o	Увеличение тени
o	Увеличение и изменение рамки фото

- Стеклянный эффект (glassmorphism) с размытым фоном
- Изменение стиля посещенных карточек через псевдокласс `:visited`
- Интерактивные элементы с реакцией на наведение
- Стилизованные кастомные SVG-иконки для каждого типа информации, встроенные непосредственно в CSS
- Градиенты и тени современного минималистичного вида
- Закругленные углы и продуманные отступы
- Цветовая схема в фиолетовых тонах с акцентами
